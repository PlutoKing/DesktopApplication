# 无人机库 UnmannedAerialVehicle

## 项目简介

- **开发目的**：无人机的相关算法
- **主要功能**：
	1. 导航、制导与控制；
    2. 自主决策、协同决策；
    3. 执行侦查、跟踪、打击、诱骗等任务。

## 原理分析

### 输入

- **操纵量**：四通道PWM，直接操纵无人机的飞行，为手动模式。
- **指令集**：任务规划系统结算出来的指令集`List<LFCommand>`，告诉无人机在某点以某姿态执行某任务。
- **信息包**：其他无人机或地面指挥系统发送来的情报，用于决策。

### 输出

- **无人机的飞行状态**：位置、姿态、速度、角速度等；
- **无人机的任务状态**：任务分配结果、任务执行情况；
